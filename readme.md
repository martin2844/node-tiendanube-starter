# TiendaNube Starter

A nodejs tiendanube bare bones starter, to start creating a TiendaNube App using:  

* Nodejs
* ExpressJs
* MongoDB
* EJS
* React (if more advanced FE is needed)
  
  
Includes:   
* Winston logger
* Mocha/Chai for testing
* Mongoose for Mongo
* Express Routing for AUTH
* Env example file
